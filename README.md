# MRP Chocolate
Tech Stack

Backend: Django

Database: SQLite3

Frontend: HTML

## Getting Started
1. Clone the repository
```bash
git clone https://github.com/sanjardvr/mrp-chocolate.git
cd mrp-chocolate
```

2. Build and run with Docker
```bash
docker build -t mrp-chocolate .
docker run --rm -p 8000:8000 mrp-chocolate
```

The application will be available at:

http://localhost:8000

Application Access
Admin Panel

URL:
